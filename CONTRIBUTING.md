# Contributing Guidelines

This repository operates under the governance of the
Hollow House Institute.

Governance authority:
https://github.com/hollowhouseinstitute/Hollow_House_Institute

## Permitted Contributions
- Documentation improvements
- Research extensions within scope
- Dataset additions following protocol
- Analysis using approved methods

## Prohibited Contributions
- Structural reordering
- Undocumented theory changes
- Anthropomorphic claims
- Removal of safeguards

## Process
1. Open an issue
2. Describe scope
3. Submit PR with references
